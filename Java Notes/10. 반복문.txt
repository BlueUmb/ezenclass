1. 반복문
    1) 어떤 조건이 성립되는 동안 반복 처리를 실행하는 제어문
    2) for문을 이용한 반복
       while문을 이용한 반복
       do-while문을 이용한 반복

2. for문
    1) for문 문법
            ①        ②       ④
        for(초기화식; 조건식; 증감식){
            ③
            수행문(실행 코드);
            ...
        }
            - 첫번째 루프의 흐름
                - ① → ② → ③ → ④
            - 두번째 루프의 흐름
                - ② → ③ → ④
            - 세번째 루프의 흐름
                - ② → ③ → ④
            - n번째 루프의 흐름
                - ② → ③ → ④
    2) for문 요소의 생략, 응용
        - 초기화식 생략가능
		    - for문 이전에 값이 초기화 되어 for내부에서 값을 지정할 필요가없을경우 생략가능
        - 조건식 생략가능
            - 반복 수행에 대한 조건이 수행문 내부에 있는경우
        - 증감식 생략가능
            - 증감식에 대한 연산이 복잡하거나 다른 변수의 연산결과값에 결정되는경우 수행문 내부에 증감식을 사용하여 생략가능
        - 위에있는 세가지의 조건이 모두 만족하면 ()안에 조건이없어도됨.
    3) 중첩 반복문
        - 반복문이 여러 번 포개진 중첩 반복
        - 반복문 내부에 또 다른 반복문이 있음
        - 여러 겹으로 반복문을 겹쳐서 구현 가능
        - 외부 반복문과 내부 반복문 간의 변수 값의 변화에 유의하여 구현해야함.
